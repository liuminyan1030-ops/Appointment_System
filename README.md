# OOP Appointment Management Utility

## Project Overview
An object-oriented scheduling system designed to manage various appointment types with a focus on calendar organization.

## Key Features
* **OOP Design**: Utilizes classes to define appointment types and scheduling logic.
* **Weekly Calendar**: Provides a structured view for retrieving and managing appointments.
* **Dynamic Calculation**: Automatically calculates fees based on service duration.

## Tech Stack
* **Language**: Python
* **Concepts**: Object-Oriented Programming, File Persistence
